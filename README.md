# TokenUsageExample

An example Xcode project demonstrating token usage tracking with Apple's Foundation Models framework in iOS 26.4.

Companion project for the blog post: [Tracking token usage in Foundation Models](https://www.artemnovichkov.com/blog/tracking-token-usage-in-foundation-models).

## Requirements

- Xcode 26.4 beta
- iOS 26.4 / macOS 26.4
- Device with Apple Intelligence enabled

## Overview

The project uses Xcode Playgrounds to show how to:

- Get the model's context size
- Measure token usage for instructions and tools
- Measure token usage for prompts
- Track transcript token usage after a conversation
- Compare token counts against the context window
